# Flashlight_Vela

Redmi Watch W6 / Xiaomi Vela 平台手电筒应用。

## 功能介绍

这是一款为小米 Vela 智能手表开发的手电筒应用，支持三种照明模式：

- **白光模式**：全屏白色高亮，支持手腕翻转自动切换最高亮度。
- **红光模式**：全屏红色，适合夜间低干扰照明。
- **白光闪光模式**：白色屏幕周期性闪烁，可作警示灯或 SOS 信号。

## 安装方法

1. 在 Release 页面下载 `Flashlight_v1.0.0.rpk`。
2. 使用小米 AIoT-IDE 或 sideload 工具将 RPK 安装到手表。
3. 安装完成后，在手表应用列表中打开“手电筒”即可使用。

## 使用说明

- **切换模式**：在屏幕上上下滑动，可在白光、红光、闪光三种模式间切换。
- **亮度控制（白光模式）**：手腕向外翻转（类似看表动作）自动提升到最高亮度；转回正常角度恢复系统亮度。
- **模式指示点**：屏幕右侧有三个定位点，显示当前所处模式。
- **退出应用**：应用会自动恢复系统亮度并关闭屏幕常亮。

## 技术要点

- 使用 `@system.brightness` 控制屏幕亮度和常亮。
- 使用 `@system.sensor` 订阅加速度计，实现手腕翻转检测。
- 使用 `<scroll>` 组件配合 `scroll-snap-type` 实现上下滑动切换模式。
- 采用 `border-radius` 适配圆形表盘，避免四角露出黑边。

## 注意事项

- 本项目使用 Vela 快应用框架开发，需要 `system.brightness` 和 `system.sensor` 权限。
- 部分手表系统版本可能不支持亮度调节 API，应用会自动降级处理。
- 长时间高亮可能导致发热和耗电，请根据实际需要使用。
